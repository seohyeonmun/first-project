# 교통봇 : 음성 기반 대중교통 안내 서비스
---
## 1. 프로젝트 소개 
- 프로젝트 주제  
  **음성 기반 실시간 버스 도착 안내 서비스**  
- 프로젝트 선정 배경  
1. 출근 시간대 빠른 정보 확인 필요  
2. 시력 불편한 사용자 화면 접근 한계  
- 기획 의도  
  호출어 + 음성  질문만으로 버스 도착 정보 제공  
---
## 2. 개발 기간
- 2026.04.16 ~ 2026.04.29
---
## 3. 팀원 소개
| 이름 | 역할 |
|---|---|
| 김현진 | KWS(openWakeWord) 파인튜닝 |
| 문서현 | STT(Whisper) + Open API(공공데이터포털) 연동 |
| 김가람 | TTS(GPT-SoVITS v2pro) 기반 음성 응답 구현 |
---
## 4. 주요 기능 ( -> 이 부분 정리 필요!!)
- KWS(openWakeWord)
- STT(Whisper)
- TTS(GPT-SoVIT)
- (예시처럼 표로 정리할 예정)
| 기능 | 설명 |
|---|---|
