# 🚌 음성 기반 버스 도착 안내 시스템

## 📌 프로젝트 소개

사용자가 음성으로 버스 번호와 정류소를 말하면,
STT(음성 인식) 기술과 공공데이터 API를 활용하여
해당 버스의 도착 예정 시간을 안내하는 시스템입니다.

예시:

> "학동증심사입구역 운림 54번 버스 언제 와?"

→ 학동증심사입구역 정류소의 운림 54번 버스 도착 시간 조회

---

## 🎯 개발 목적

* 음성 인터페이스 기반 교통 정보 제공
* 대중교통 정보 접근성 향상
* Whisper STT 실전 활용
* 공공데이터 API 연동 경험

---

## ⚙️ 사용 기술

* Python
* Whisper (OpenAI Speech-to-Text)
* Requests
* XML Parsing
* Levenshtein Distance
* 광주광역시 버스 공공데이터 API

---

## 🔄 시스템 흐름

음성 입력
→ Whisper 음성 인식
→ 버스 번호 추출
→ 정류소 이름 추출
→ 유사 정류소 검색
→ 버스 도착 정보 조회
→ 결과 출력

## 🧠 핵심 기능

### 1️⃣ 음성 인식

사용자 음성을 텍스트로 변환합니다.

### 2️⃣ 버스 번호 추출

정규표현식을 통해 버스 번호를 인식합니다.

### 3️⃣ 정류소 추출

문장에서 정류소 이름 후보를 추출합니다.

### 4️⃣ 유사도 비교

Levenshtein 알고리즘으로 가장 유사한 정류소를 찾습니다.

### 5️⃣ 도착 시간 안내

해당 버스의 남은 도착 시간을 반환합니다.
